So, you want to know how to use this SDK?

Here is some sample documentation for basic usage. These sample lines should work from either the interactive interpreter or a script.

Initial setup

>>> from familysearch import FamilySearch
>>> app_key = "MY-MAGIC-DEV-KEY-THAT-ALLOWS-FAMILY-SEARCH-ACCESS" # You didn't think I'd give you a free one, now did you?
>>> user_agent = "MyApp/VersionNumber"
>>> fs_address = "https://sandbox.familysearch.org"
>>> fs = Familysearch(user_agent, app_key, base=fs_address)

As a side note: I left the FamilySearch base configurable so that, if there is a need to change it to, say, "https://deployment.familysearch.org" (which has happened for temporary sandbox upgrade), it is available.

Logging in

Desktop login

The UN/PW login method is recommended for desktop clients, and is, in fact, required for certification.

>>> try:
...     input = raw_input

>>> from getpass import getpass
>>> fs.login(input("Username: "), getpass())
